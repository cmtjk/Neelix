Known Problems and Workarounds

Warum laden manchmal Tabs nicht?
Weil das das Netzwerk wieder "rumspackt"(Annette). Es wird dennoch weiter versucht, die Datenbank zu kontaktieren und eigentlich ist es so programmiert, dass es nach einiger Zeit mit einer pompösen Fehlermeldung kundtut, dass irgendwas nicht stimmt. Im Zweifelsfall hilft ein Programmneustart.

Warum dauert es so lange bis eine Anfrage gespeichert wurde oder die Bestätigung erscheint?
Gleicher Grund wie oben, aber vielleicht liegts auch daran, dass jetzt für jede Anfrage die Verbindung neu erstellt wird. Das wird sich in einer zukünftigen Version auch wieder ändern, sobald das Netzwerk wieder stabil ist.
