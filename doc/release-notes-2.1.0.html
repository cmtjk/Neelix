Product Release Information

Product: Neelix (working title)
Release Number: 2.1.0
Release Date: 06-09-2016

Introduction

Fast keine dramatischen Änderungen, außer dass der ganze alte SQLite-Kram rausgeflogen ist und Neelix nun problemlos vom eigenen PC ausgeführt werden kann und soll.

What's New?

  • removed SQLite implementation
  • added notification if new version is available
  • optimized some queries
